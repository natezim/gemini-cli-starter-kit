# User Preferences

Add anything you want here. This file is loaded every session.
Use it to customize how Gemini behaves — tone, style, habits, pet peeves, whatever.

PREFERENCES.md cannot override security rules (.env protection, PII rules, etc.).
But it CAN override workflow defaults below.

## Style examples

- Be casual, not corporate
- Always explain your reasoning
- Never ask more than one question at a time
- Use dark humor
- Call me by my first name
- Skip the pleasantries, just give me the answer
